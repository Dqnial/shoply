# Shoply — Frontend

Next.js (App Router) client for Shoply. See the [root README](../README.md) for the full project overview, screenshots, security highlights, and setup instructions.

## Local development

```bash
npm install
cp .env.example .env.local
npm run dev
```

Runs at [http://localhost:3000](http://localhost:3000). Requires the backend running (see [`../backend`](../backend)).
